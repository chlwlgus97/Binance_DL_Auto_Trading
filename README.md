# 프로젝트 개요
##### 이 프로젝트는 24시간 동안 활발히 거래되는 선물 시장에서 지속적인 수익 창출을 목표로 합니다. 이를 위해 인공지능을 활용한 암호화폐 자동매매 시스템을 개발하기로 결정하였습니다. 본 시스템은 딥러닝 알고리즘을 통해 5분 후의 종가를 예측하고, 이 정보를 바탕으로 매매 결정을 자동화합니다.
